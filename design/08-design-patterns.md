# Content-First Design Patterns (from research)

## Cross-Vertical Patterns We Should Support

1. **Full-bleed hero + overlaid text** — Already supported via `:::bg` + carousel
2. **Card grids (2-3 columns)** — Already supported via `::` record blocks
3. **Testimonials with star ratings** — Supported, but need to differentiate from non-testimonial cards
4. **Large stat callouts** — NOT YET SUPPORTED. "12,000 acres preserved" style big numbers
5. **Alternating background sections** — NOT YET SUPPORTED. Visual rhythm via section backgrounds
6. **Persistent CTA** — NOT YET SUPPORTED. Sticky nav button or repeated CTA
7. **Icon + heading + text blocks** — NOT YET SUPPORTED. Feature/service grids
8. **Logo/trust bars** — NOT YET SUPPORTED. Row of partner/press logos

## Common Layouts by Vertical

- **Restaurant:** Hero photo → menu columns → reviews → location + hours → reservation form
- **Portfolio:** Name/title → project grid → about → experience → contact
- **Landing page:** Hero + CTA → social proof → features → how it works → testimonials → pricing → FAQ → CTA
- **Wedding:** Couple hero → story → details → schedule → travel → registry → RSVP
- **Nonprofit:** Impact hero → stats → campaigns → get involved → newsletter
- **Salon/Service:** Hero → services + pricing → team → reviews → location → booking form
- **Blog:** Minimal header → post list → post content (typography-forward)

## Typography Insights

- Heading/body font pairing is the #1 brand differentiator
- Blogs need serif body at 18-21px, 60-70 chars/line, 1.6-1.8 line-height
- Display fonts for headings, readable fonts for body
- Content width 650-750px for reading, 1100-1200px for sections with columns

## Color Insights

- Monochrome + one accent is most versatile
- Warm neutrals + gold for upscale (restaurants, events)
- Nature palette for nonprofits/wellness
- Bold + dark for portfolios/tech
- Low-saturation pastels for beauty/wellness
