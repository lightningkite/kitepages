One interface, everything you need. Raycast replaces dozens of apps and context switches with a single keyboard shortcut.

```
> Fix spelling and grammar
> Translate to Spanish
> Explain this code
> Make this email more professional
> Summarize in 3 bullet points
```

## Don't repeat yourself

Automate the things you do all the time. Raycast gives you three powerful tools to eliminate repetitive work.

|||
::: card
### Snippets

Tired of typing the same thing? Create a snippet and insert it by typing its keyword.

```
;addr → 3rd Floor, 1 Ashley Road
        WA14 2DT Altrincham, Cheshire

;sig  → Best regards,
        Alex Thompson
        alex@company.com
```
:::
---
::: card
### Quicklinks

Say goodbye to open tabs. Create quicklinks to launch anything from anywhere.

```
gh {query} → github.com/search?q={query}
jira {id}  → company.atlassian.net/browse/{id}
docs {q}   → docs.google.com/search?q={q}
```
:::
---
::: card
### Hotkeys & Aliases

Speed up your workflow by assigning hotkeys or aliases to common commands.

- [[⌥]] [[Space]] — Clipboard History
- [[⌘]] [[⇧]] [[M]] — Toggle Microphone
- [[⌘]] [[⇧]] [[D]] — Toggle Do Not Disturb
:::
